# 📊 Análisis de Ventas - Challenge Data Science Latam

Este proyecto consiste en un análisis detallado de cuatro tiendas de retail para determinar la mejor oportunidad de negocio para el Sr. Juan. Se utilizaron técnicas de manipulación de datos y visualización para extraer insights clave.

## 🎯 Objetivo
Evaluar el desempeño de 4 tiendas basándose en ingresos, satisfacción del cliente, volumen de ventas por categoría y costos logísticos.

## 🛠️ Herramientas Utilizadas
* **Python 3**
* **Pandas**: Para la limpieza y manipulación de datos.
* **Matplotlib / Seaborn**: Para la generación de gráficos estadísticos.
* **Google Colab**: Entorno de desarrollo en la nube.

## 📈 Hallazgos Clave
En este análisis se cubrieron los siguientes puntos:
1. **Ingresos Totales:** Cálculo del volumen de ventas bruto por tienda.
2. **Categorías Populares:** Identificación de los productos más demandados.
3. **Satisfacción del Cliente:** Análisis de calificaciones promedio.
4. **Logística:** Costo promedio de envío pagado por el cliente.
5. **Top/Bottom Productos:** Listado de productos con mayor y menor rotación.

## 🚀 Conclusión del Proyecto
Tras cruzar las métricas de rentabilidad y satisfacción, la recomendación final fue enfocarse en la **Tienda [X]**, debido a su balance óptimo entre ingresos y lealtad del cliente.
